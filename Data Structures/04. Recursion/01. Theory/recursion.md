## Recursion:

- Function calling itself directly or indirectly is called recursion and the corresponding function is called recursive function.

- Types of recursion
  - Head recursion
  - Tail recursion

</br>

## Recursion structure:

```
 ...fun(...){
     base case
     ...

     Recursive call (i.e. call to fun())
     with at least one change in parameter
 }

```
