## Hashing

- Used to store `key : value` pairs
- Best for `Search, Insert, Delete` operations O(1)

- Not useful for:

  - Finding closest value
  - Sorted data
  - Prefix Searching

- Applications:
  - Dictionaries
  - Database Indexing
  - Cryptography
  - Caches
  - Symbol table
  - Routers
  - Getting data from databases
